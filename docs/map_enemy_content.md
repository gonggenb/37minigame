# 主地图区域与敌人内容表

## 当前目标

主地图保持 44 × 38 的短局探索尺度，通过区域地标、边缘装饰和不同敌人定位提升精细度，而不是继续扩大空旷面积。

## 区域结构

| 区域 | 视觉识别 | 主要敌人方向 |
| --- | --- | --- |
| 中央驿路 | 十字土路、桥、集市、村舍 | 山贼、巨鼠、刀客 |
| 东郊村落 | 房屋、果树、围墙、村门 | 快剑、机关傀、流寇 |
| 北岭关隘 | 城墙、瞭望塔、松树、巨石 | 山魈、黑衣客、机关车 |
| 西林废道 | 密林、溪桥、遗迹、商队 | 毒客、妖狼、重弩 |
| 南矿山路 | 矿洞、寨墙、采石堆、关楼 | 毒刃、赤骑、岩窟守卫 |

地图装饰遵守 [2.5D 武侠绘本美术规范](art_style_guide.md)：场景保持低对比和低反射，敌人轮廓优先于背景细节。

## 密度控制

- 道路中心和遭遇触发器周围保留清楚的移动空间。
- 树木、岩石和墙体优先布置在区域边缘与道路转折处。
- 模型只承担视觉作用，不增加复杂 MeshCollider。
- 后续新增敌人优先替换重复敌人，而不是持续增加总密度。
